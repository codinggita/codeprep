# CodePrep 🚀

**Track. Practice. Improve.**

CodePrep is a web application that helps developers—especially new learners—track their coding interview preparation by organizing solved problems, revision schedules, and notes in one place.

It automatically fetches solved problems from coding platforms and keeps them updated periodically, helping users monitor their progress and stay consistent with their preparation.

CodePrep aims to be an **all-in-one preparation hub for developers** solving problems across multiple platforms.

---

# 🚀 Purpose

CodePrep was built to:

* Improve and showcase full-stack development skills
* Provide a useful tool for developers preparing for coding interviews
* Help learners stay organized while solving problems across multiple platforms

---

# 👨‍💻 Target Users

CodePrep is designed for:

* Beginners learning **Data Structures & Algorithms**
* Developers preparing for **coding interviews**
* Competitive programmers using platforms like **LeetCode** or **Codeforces**
* Anyone who wants to **track their problem-solving progress**

---

# ✨ Features

## 🔗 Multi-Platform Integration

Users can connect their coding platform accounts such as:

* LeetCode
* Codeforces
* Other coding platforms (future support)

At least **one platform connection is required**.

---

## 📥 Automatic Problem Sync

After connecting a platform account:

* CodePrep automatically fetches solved problems
* Problems are **periodically updated every 30–60 minutes**
* Keeps preparation data up-to-date without manual effort

---

## 📝 Problem Tracking

CodePrep stores and organizes solved problems with useful information:

* Problem Number
* Problem Name
* Platform
* Difficulty
* Topic
* Problem Link
* Date Solved
* Revision Date
* Notes

Users can also **manually add problems if needed**.

---

## 🎯 Daily Goals

Users can set **daily problem-solving goals** to stay consistent with their preparation.

---

## 🔍 Smart Search

Problems can be searched using:

* Problem name
* Problem number

This makes it easy to revisit previously solved questions.

---

## 📊 Performance Analysis

CodePrep provides a performance overview to help users understand their progress.

Users can view:

Weekly performance analysis

Monthly performance analysis

Overall preparation statistics

This helps developers monitor improvement and identify weak areas.

---

## 🔁 Revision Tracking

CodePrep helps users review important problems by allowing them to:

* Schedule revision dates
* Track problems that need revisiting
